1.4. Python市场

  • python为什么火

    • 由需求引发的市场

    • 相对比较简单，面对中小企业和原PHP市场还有巨大需求

  • 要不要学

    • Python是很好的教学课程，市场已经做出了选择

    • Python应用广泛，功能强悍

    • 与其做Java的牛尾，干嘛不做Python的鸡头
